# Portfolio Website

## Overview
This is a personal portfolio website built with React, showcasing projects, skills, and contact information. The site includes components for navigation, home page, about section, projects display, skills list, contact form, and footer.

## Features
- Responsive design for mobile and desktop.
- Dynamic content sections.
- Contact form integration.
- Project showcase with links.

## Technologies Used
- React.js
- HTML5
- CSS3
- JavaScript
- Framer Motion

## Installation
1. Clone the repository
   ### `git clone`
   https://github.com/yourusername/portfolio.git
2. Navigate to the project directory:

## `cd portfolio`

3. Install dependencies:

## `npm install`

4. Start the development server:

## `npm start`

5. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Usage
- Customize the content by editing the component files as listed in the changes below.
- Deploy to platforms like Netlify or Vercel for production.

## Project Structure
- `src/components/`: Contains all React components.
  - `Navbar.jsx`: Navigation bar.
  - `Home.jsx`: Landing page.
  - `About.jsx`: About section.
  - `Projects.jsx`: Projects display.
  - `Skills.jsx`: Skills list.
  - `Contact.jsx`: Contact form.
  - `Footer.jsx`: Footer section.
